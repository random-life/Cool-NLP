**数据名称：**对话数据

**数据规格**: 1.1G闲聊语料

**数据描述**：

该库搜集了包含

- chatterbot
- 豆瓣多轮
- PTT八卦语料
- 青云语料
- 电视剧对白语料
- 贴吧论坛回帖语料
- 微博语料
- 小黄鸡语料

共8个公开闲聊常用语料和短信，白鹭时代问答等语料。

并对8个常见语料的数据进行了统一化规整和处理，达到直接可以粗略使用的目的。

**数据集划分**：8个tsv问句
